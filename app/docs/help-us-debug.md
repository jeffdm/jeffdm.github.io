## Help Us Debug

---

You can start Kitematic from the command-line via the following steps:

{% highlight bash %}
cd <dir with Kitematic.app>
./Kitematic.app/Contents/MacOS/Kitematic
{% endhighlight %}

If possible please attach logs to any issues reported on GitHub.
