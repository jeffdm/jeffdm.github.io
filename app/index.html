How does it work?

Kite automatically sets your team up with container-based, portable, reproducible environments for building web services. Kite does this all in the cloud, using your favorite provider (Amazon AWS for now, more coming later), so there's no lock-in.

How does it fit into my workflow?

Kite integrates with GitHub and Docker to automatically make creating environments for any kind of project simple. Kite includes a powerful tool chain allows you to keep using your favorite editor to write code.

Teams members can snapshot their Kite environments, show off their changes as they build and have large datasets available for development instantly.
